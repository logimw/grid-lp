# grid-lp
Landing page created with grid. SCSS/gulp.js


1. Clone repo.
2. ```cd grid-lp```
3. ```npm install```
4. ```gulp-serve``` - It's gulp 4.0+, so please make sure that You have new enough npm/node version.


And You will see (soon) good looking, only grid layout.
